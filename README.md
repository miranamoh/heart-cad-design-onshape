# heart-cad-design-onshape
Custom heart part designed in Onshape, exported as STL


<img width="852" height="711" alt="heart-screenshot" src="https://github.com/user-attachments/assets/5fecb057-0668-4967-a762-d55858fd28e1" />


This task involves designing a custom 3D part using **Onshape** based on personal measurements, exporting it as an STL file, and preparing it for 3D printing.

## Overview

- **Tool used:** [Onshape](https://www.onshape.com/) 
- **Shape:** Heart
- **Thickness:** 2 mm
- **Key hole:** 1 hole
- **Export format:** STL 

## Design Process

1. **Sketch:** The heart outline was created using two equal circles for the upper lobes and two symmetric angled lines (122° each) meeting at the bottom point, with matching dimensions on both sides (44 mm, 88 mm) to ensure a fully closed, symmetric profile.
2. **Extrude:** The closed sketch was extruded to a thickness of 2 mm to create the solid part.
3. **Hole:** A single hole with a 4 mm diameter was added at the required position using a second sketch and an extrude cut.
4. **Export:** The final part was exported in STL format (Binary, Millimeter units, Fine resolution) for 3D printing.
